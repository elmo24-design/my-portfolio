Project built: Snapgram

Fullstack Project

Project development duration:

1 month and 15 days

Description:

Snapgram, basically a mini-instagram clone but it is an application that I’ve created where users can store and keep track of their memories. I created this app from scratch, starting from the design phase using figma then converting it into code and deploying it to Netlify.

Technologies Used:

Figma
React JS
Firebase
Material UI
Framer Motion
SASS/SCSS

Notable Features:

  • Authentication - Sign in / Sign up
  • Add post, Edit/Update Post, Delete Post
  • Image upload
  • Set post as private / Set post as public
  • Show post
  • Like Post
  • View users who like the post
  • Comments
  • Follow/Unfollow
  • Show followers list / Show following list
  • Notifications
  • Show List of users
  • Search for users
  • Show indication of following / being followed
  • Visit other users’ profile
  • Chatroom
  • Profile
  • Update Avatar
  • Edit Basic Info
  • Change Password

Project built: PWCC Application

Fullstack Project

Project development duration:

1 month and 15 days

Description:

PWCC Application is a Profiling and Sales Management App that I’ve created for the Pililla Women’s Consumers Cooperative in the Municipality of Pililla, Rizal to help them manage their transactions.
I created this app from scratch, starting from the design phase using figma, converting it into code and then deploying it to Netlify.

Technologies Used:

Figma
React JS
Firebase
Material UI
Framer Motion
SASS/SCSS

Notable Features:

  • Authentication - Sign in
  • Add, Edit/Update, Delete Officers/Members
  • Archive Officers/Members
  • Restore Officers/Members
  • Move Officers to Members’ list
  • Move Members to Officers’ list
  • Add, Edit/Update Capital Shares
  • Dashboard showing total of officers, members and Capital Shares
  • Add date for a particular product
  • Add, Edit/Update, Delete items
  • Show Sales Report
  • Convert Sales Report to PDF File
  • Search for Officers/Members

Project built: CMS App

Fullstack Project

Project development duration:

1 month

Description:

CMS App, A Client Management System I’ve built for Mosfetech digital electronics services to help them manage their clients.
I created this app from scratch, starting from planning (sketching out the app), converting it into code and then deploying it to Netlify.

Technologies Used:

Vue JS (Vue 2)
Firebase
Vuetify

Notable Features:

  • Authentication - Sign in
  • Register new user - admin side
  • Add, Edit/Update, Delete Clients
  • Search for clients
  • Show indication (ongoing, done, delivered)
  • Add, Edit/Update, Delete Expenses
  • Archive Clients
  • Restore Clients
  • Print Clients’ receipt
  • Convert Clients’ list to PDF file
  • Dashboard showing total of clients, total sales and expenses
  • Light Mode / Dark Mode
